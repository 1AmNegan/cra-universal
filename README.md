# Create React App Universal CLI

Simple express server for your Create React App projects with Server-side rendering and Code-splitting

## How to use
```sh
# Install global CLI
npm install -g create-react-app cra-universal

# Init CRA
create-react-app my-app

# Init CRA server
cd my-app/
cra-universal init

# Bundle your client
npm run build

# Start server
cd ./server
npm install
npm start

```
